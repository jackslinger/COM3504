Tracking Discussions

This page is designed to allow you to retrieve upto 100 tweets about a given topic within a set number of days. This is one of the simplest parts of the site with sends the data from the form to twitters search API and returns the tweets recieved.

The From field is optional and causes the system to look only for tweets from a particular twitter user. The Latitude and Longitude fields are optional, if both are filled in, the search returns only tweets within 5 miles of that location by useing the tweets geocode.
